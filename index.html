    How set up responsive Nav Bar?

    1. Start with basic markup for the list of items to navigate to.
    2. Style the nav with a background, then the menu with the list style set to none, and a flexbox, justifying to the end and aligning links to the center.
    3. Declare a height for the anchor tags, list items match the anchor height. The children will define the height of the parent nav bar.
    4. Apply inline padding to the links, remove default text decoration from anchor tags if customizing. Set the parent elements as display flex and align links to the center.
    5. Put the logo nav element to the left by selecting the first child of the nav bar, and setting automatic margin to the right.
    6. Copy the list of navigation links into another menu with the class sidebar. Style the sidebar to be removed by default, toggleable by resize or button.
